📋 项目概述

Log OpenCode 是一个基于 Python 的 AI 驱动日志分析工具，利用大语言模型（Qwen3-235B）智能分析日志文件并生成结构化报告。项目提供 CLI 和 Web 两种使用方式，支持异步处理、分块分析和多种输出格式。

🧩 模块职责

模块 文件 职责描述
日志解析器 log_parser.py 解析多种格式日志，提取时间戳、级别、消息
AI 分析器 log_analyzer.py 调用大语言模型分析日志，支持云端/本地模型降级
Web 后端 app.py 提供文件上传、分析任务、进度查询、报告下载 API
CLI 入口 main.py 命令行界面
任务日志 TaskLogger 任务执行日志（按日期分目录）
一键启动 start.sh 自动启动 Ollama + Web 后端

📊 10 步流水线

┌─────────────────────────────────────────────────────────────────────┐ │ 分析任务 10 步流水线 │ ├─────────────────────────────────────────────────────────────────────┤ │ Step 1: 文件上传 (已完成，由前端上传) │ │ Step 2: 文件去重校验 → MD5 校验文件完整性 │ │ Step 3: 加载配置 → 读取 config.yaml │ │ Step 4: 解析日志 → LogParser.parse_file() │ │ Step 5: 条目统计 → 计算级别分布 │ │ Step 6: 分块策略 → 按 batch_size 分块 │ │ Step 7: AI 模型分析 → 逐块调用 AI → 合并结果 │ │ Step 8: 生成 Markdown → 保存到 output/{task_id}.md │ │ Step 9: 生成 PDF → 保存到 output/{task_id}.pdf（支持中文） │ │ Step 10: 完成 → 更新任务状态，返回结果 │ └─────────────────────────────────────────────────────────────────────┘

🆕 v1.6 版本新增/改进

1. 任务中止功能 ⏹️

前端 停止按钮 + stopAnalysis() 函数
后端 POST /api/abort/{task_id} 端点
执行检测 分析循环中检查 task.get('aborted', False)

2. 历史记录功能 📜

端点 GET /api/history
返回数量 最多 20 条
筛选条件 status in ['completed', 'error']
排序 按 task_id 倒序（最新优先）

3. 预估剩余时间 ⏱️

记录每块处理时间 chunk_times_ms，计算平均处理速度，动态预估剩余时间。

⚙️ 配置项说明

config.yaml 结构

model:                    # AI 模型配置
  provider: "modelarts"   # 提供商: modelarts / ollama
  name: "qwen3-235b-a22b"# 模型名称
  base_url: "https://..." # API 端点
  api_key: "..."         # API 密钥
  fallback_provider: "ollama"     # 降级提供商
  fallback_model: "qwen3.5:9b"    # 降级模型
  fallback_url: "http://localhost:11434/v1"

log:                     # 日志解析配置
  formats:               # 支持的日志格式
    - name: "standard"
    - name: "apache_common"
    - name: "json"
  levels: [DEBUG, INFO, WARNING, ERROR, CRITICAL]

analysis:                # 分析配置
  batch_size: 5000      # 单次最大分析条目数

output:                  # 输出配置
  format: "markdown"
  directory: "../output"

opencode:                # OpenCode 集成（待实现）
  enabled: true
  auto_launch: false

🚀 使用方式

一键启动（推荐）

cd /Users/a666/Documents/trae_projects/log_opencode
